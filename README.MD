# Quito Coffee Roasters - Dashboard Interactivo ☕

Aplicación web interactiva desarrollada para la tostadería artesanal de café de especialidad "Quito Coffee Roasters", ubicada en el Centro Histórico de Quito. Este proyecto fue desarrollado como parte de la Ruta de la Empleabilidad de ConQuito.

## 🚀 Características del Proyecto
- **Catálogo Dinámico:** Filtrado por origen (Ecuador, Colombia, Brasil, Etiopía) y búsqueda en tiempo real de cafés especiales.
- **Calculadora Interactiva de Pedidos:** Carrito de compras interactivo que actualiza subtotal, costo de envío según la zona de Quito y cálculo del total en tiempo real.
- **Integración de APIs Públicas:**
  - **Open-Meteo API:** Clima actual de Quito en tiempo real.
  - **Official Joke API:** Genera frases divertidas sobre café dinámicamente.
  - **Exchange Rate API:** Conversión en un clic de USD a EUR (€).
- **Formulario Validado:** Validación interactiva en tiempo real con mensajes de error para Nombre, Email y Teléfono.
- **Diseño Responsive:** Layout adaptado perfectamente a teléfonos móviles, tabletas y computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 Moderno (CSS Grid, Flexbox, Variables de Diseño)
- JavaScript Moderno (ES6+) con estructura de módulos asíncronos (ESM)
- FontAwesome Icons
