# Registro de Uso y Co-Piloto de IA 🤖

En este documento se evidencia el uso de asistentes de IA para acelerar el desarrollo del Dashboard interactivo.

## 💡 Top 3 Prompts Más Efectivos
1. *"Escribe una estructura HTML5 semántica y accesible utilizando etiquetas CSS Grid y Flexbox inspiradas en colores tierra y café para una tienda de café llamada Quito Coffee Roasters."*
2. *"Crea un script en JS moderno modular que consuma de forma asíncrona mediante fetch y async/await el clima actual de Quito con la API Open-Meteo."*
3. *"Genera una función de validación de formularios en tiempo real con expresiones regulares para nombres, teléfonos de Ecuador y correo electrónico."*

## 📝 Reflexión sobre el Aprendizaje
El uso de asistentes de IA actuó como un mentor personalizado en tiempo real. Me permitió acelerar la estructuración modular de los scripts en archivos separados (import/export) y comprender mejor cómo estructurar el manejo seguro de errores con bloques `try/catch` al consumir servicios asíncronos externos. Ahorré tiempo de diseño CSS gracias a las sugerencias de la paleta corporativa y pude concentrarme en pulir la experiencia de usuario y la lógica de la calculadora de pedidos en Quito.
